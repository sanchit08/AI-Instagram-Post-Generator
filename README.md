# AI-Instagram-Post-Generator
A multi-agent AI system that generates Instagram posts using LLMs and image generation models.

🚀 Features

✍️ Caption generation (LLM)

🎨 Image prompt generation (LLM)

🖼️ Image generation (OpenAI)

💾 Local storage of posts & images

📱 Instagram-style preview UI

🤖 Multi-agent architecture

🧩 Architecture
User Input
   ↓
Caption Agent (LLM)
   ↓
Image Prompt Agent (LLM)
   ↓
Image Generator (DALL·E)
   ↓
Storage + UI Preview
